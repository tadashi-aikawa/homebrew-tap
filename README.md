# homebrew-tap

tadashi-aikawa 製ツールの Homebrew tap。

## 使い方

```bash
brew install tadashi-aikawa/tap/jinrai
```

## 提供パッケージ

| 名前 | 説明 |
| --- | --- |
| [jinrai](https://github.com/tadashi-aikawa/jinrai) | 思考の速度で素早くウィンドウ操作を行う macOS 用ツール |

> [!NOTE]
> cask のバージョンと sha256 は [jinrai の Release ワークフロー](https://github.com/tadashi-aikawa/jinrai/blob/main/.github/workflows/release.yml)から自動更新されます。
